Bonus Grasshoper

GRASSHOPER

Paso a paso:

  • PASO 1:

Creamos un bloque que contenga la pieza que queremos mecanizar, en este caso la figura realizada en grasshoper. Anteriormente cuidamos que esta estuviera ortogonal, para poder realizar el mecanizado de forma correcta.

  • PASO 2:

Luego de esto, vamos a hacer la primera operación de desvaste, para remover el material excedente.

  • PASO 3:

Ahora realizaremoms un recorrido en paralelo en 1 eje, para de esta forma limpiar la figura del material excedente.

  • PASO 4:

Luego duplicaremos la operación anterior, pero esta vez la rotaremos en 90 grados, para generar el recorrido en el eje contrario.

  • PASO 6:

Ahora vamos a visualizar el recorrido completo de este mecanizado, considerando las 3 operaciones que realizamos anteriormente: desvaste, paralelo en 1 eje, paralelo en 2 eje.
